# Available Scripts

## Build the library

``` bash
npm run rollup
```

## Publish the library

``` bash
npm publish
```

## Run tests

``` bash
npm run test
```

## Run storybook locally

``` bash
npm run storybook
```

## Build storybook

``` bash
npm run build-storybook
```

## To install library for use

``` bash
npm install @iwunduchuks/invasion-ui
```
